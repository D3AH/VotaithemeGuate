============
Installation
============


Stable release
--------------

To install Votainteligente Theme Red Ciudadana, run this command in your terminal:

.. code-block:: console

    $ pip install votainteligente_theme_red_ciudadana

This is the preferred method to install Votainteligente Theme Red Ciudadana, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Votainteligente Theme Red Ciudadana can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/D3AH/votainteligente_theme_red_ciudadana

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/D3AH/votainteligente_theme_red_ciudadana/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/D3AH/votainteligente_theme_red_ciudadana
.. _tarball: https://github.com/D3AH/votainteligente_theme_red_ciudadana/tarball/master
